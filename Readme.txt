
------------------Hướng dẫn cài đặt backend-----------------
B1 : Cài Nodejs
B2 : vào server/src
B3 : Mở cửa sổ lệnh ở thư mục này
B4 : Chạy lệnh npm i
B5 : Chạy lệnh npm i nodemon --global
B6 : Cài MySql trên server
B7 : restore database với file "aloxe.sql"
B8 : Cấu hình username, password để connect vào database trong file .env hoặc src/config/init.js
B0 : Chạy lệnh npm run dev

------------------Hướng dẫn cài đặt font-end-----------------
B1 : Cài Nodejs

B3 : Mở cửa sổ lệnh ở thư mục này
B4 : Cấu hình app_mobile_client/hcdh-tpkm/src/environment.ts
B5 : Chạy lệnh npm i
